# 데이터 시각화 시뮬레이터 MVP

브라우저에서 서버 없이 데이터를 파싱하고 여러 차트 후보를 즉시 보여주는 로컬 우선 시각화 시뮬레이터입니다.

## 핵심 범위

- 샘플 데이터 선택
- 직접 입력/붙여넣기
- 로컬 CSV 파일 처리
- 컬럼 타입 추론
- 여러 차트 후보 카드 표시
- 부적합 차트의 더미/경고 상태 표시
- 대량 차트의 최근 구간 확대와 상위값 필터

## 제외 범위

- 서버 업로드
- 로그인/프로젝트 저장
- 코드 스니펫 또는 차트 설정 내보내기
- 대시보드 편집기

## 데이터 처리 제한

- 데이터는 브라우저에서만 처리됩니다.
- 기본 제한은 1MB 또는 5,000행입니다.
- MVP CSV 지원 범위는 단순 CSV/TSV 중심입니다.


## UX 고도화 방향

현재 버전은 먼저 큰 차트를 보여주고, 필요한 입력과 데이터 확인은 주변 패널에서 빠르게 처리하는 흐름을 지향합니다.

1. 상단 입력 영역에서 샘플 CSV, 파일 선택, 붙여넣기 중 하나로 데이터 불러오기
2. 메인 차트 영역에서 선택한 차트를 크게 확인
3. 아래 후보 버튼으로 다른 차트 표현 비교
4. 컬럼 요약과 미리보기로 데이터 해석 확인
5. 지점이 많은 차트는 `최근 12개 확대` 또는 `상위 12개 필터`로 표시 범위를 좁혀 확인

차트 후보는 `바로 보기`, `확인 필요`, `조건 부족` 상태로 나뉘며, 맞지 않는 차트도 숨기지 않고 이유를 알려줍니다.
