# Chart Deck

브라우저에서 CSV를 바로 읽고, 데이터에 맞는 차트 후보를 비교할 수 있는 **로컬 우선 데이터 시각화 워크벤치**입니다.

서버 업로드 없이 데이터를 확인하고, 차트 종류·기준 컬럼·값 컬럼·날짜 집계 방식을 바꿔 보면서 데이터에 맞는 관점을 빠르게 찾는 데 초점을 맞췄습니다.

## 데모 성격

- **입력**: 샘플 데이터, 로컬 CSV 파일, 텍스트 붙여넣기
- **분석**: 컬럼 타입 추론, 행/컬럼 정보 요약
- **시각화**: 여러 차트 후보 구성 및 비교
- **상호작용**: 기준/값 컬럼 선택, 날짜 표시 방식 선택, Recharts Tooltip/Brush 기반 범위 탐색
- **테마**: 라이트 / 다크 모드 전환

## 핵심 기능

### 1. 로컬 우선 데이터 입력
- CSV 파일 선택
- CSV 텍스트 붙여넣기
- 샘플 데이터 즉시 로드
- 브라우저 내부 처리, 서버 업로드 없음

### 2. 차트 후보 비교
데이터 컬럼을 분석해 아래 차트 후보를 구성합니다. 첫 후보는 자동으로 보여주지만, 사용자가 직접 다른 표현 방식과 값 컬럼을 선택할 수 있습니다.

- 선 차트
- 영역 차트
- 막대 차트
- 산점도
- 파이 차트
- 도넛 차트
- 레이더 차트

각 차트는 단순히 숨기지 않고 아래 상태로 보여줍니다.

- **바로 보기**: 지금 데이터로 자연스럽게 표현 가능
- **대체 기준**: 날짜/범주가 부족해 행 순서 같은 대체 기준 사용
- **데이터 부족**: 필요한 컬럼이 부족함

### 3. 표시 기준 선택
- 숫자 컬럼이 여러 개면 차트에서 볼 값을 선택할 수 있습니다.
- 막대/파이/도넛/레이더 차트는 기준 컬럼을 바꿔 볼 수 있습니다.
- 날짜 컬럼이 있는 차트는 원본 순서를 기본으로 유지합니다.
- 필요할 때만 일별 / 월별 / 연도별 집계를 선택합니다.
- 집계 계산은 합계와 평균 중 선택할 수 있습니다.

### 4. 대량 데이터 대응
- 전체 데이터를 기준으로 차트를 유지
- 5,000행 제한 안에서는 전체 지점을 그대로 렌더링
- 선/영역/막대 차트는 Recharts Brush로 구간을 좁혀 볼 수 있음
- 페이지 스크롤과 충돌하던 직접 구현 휠 줌/버튼 토글은 제거
- 파이/도넛 차트는 비중 비교가 흔들리지 않도록 구간 확대를 적용하지 않음

### 5. 데이터 이해 보조 패널
- 행 수 / 컬럼 수 / 감지된 컬럼 타입
- 컬럼 타입 추론 결과
- 표 미리보기

## 상호작용 설계 메모

기존에는 차트 확대와 드래그를 직접 SVG 위에 구현했지만, 버튼 토글과 오버레이가 늘어나면서 사용 흐름이 복잡해졌습니다. 현재는 차트 렌더링과 기본 상호작용을 Recharts에 맡기고, 앱 UI는 다음 선택지만 남겼습니다.

- 어떤 차트로 볼지
- 어떤 컬럼을 기준/값으로 쓸지
- 날짜를 원본/일별/월별/연도별 중 어떻게 볼지
- 대량 데이터에서 Brush로 어느 구간을 볼지

## 데이터 처리 제한

- 브라우저에서만 처리됩니다.
- 기본 제한은 **1MB 또는 5,000행**입니다.
- 단순 CSV / TSV 중심 입력을 가정합니다.

## 프로젝트 의도

이 프로젝트는 단순 차트 뷰어보다,

> **“데이터를 넣고 표현 방식과 집계 기준을 직접 바꿔 보며 빠르게 판단할 수 있는 UI”**

를 만드는 데 목적이 있습니다.

즉, 차트를 그리는 것 자체보다도  
**입력 → 후보 비교 → 표시 기준 선택 → 해석 확인** 흐름을 짧고 명확하게 만드는 데 집중했습니다.
