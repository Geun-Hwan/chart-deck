# Chart Deck

브라우저 안에서 CSV를 바로 읽고, 데이터에 맞는 차트를 추천하고 비교할 수 있는 **로컬 우선 데이터 시각화 워크벤치**입니다.

서버 업로드 없이 빠르게 데이터를 확인하고, 여러 차트 표현을 바꿔 보면서 어떤 차트가 가장 잘 맞는지 판단하는 데 초점을 맞췄습니다.

## 데모 성격

- **입력**: 샘플 데이터, 로컬 CSV 파일, 텍스트 붙여넣기
- **분석**: 컬럼 타입 추론, 데이터 상태 요약
- **시각화**: 여러 차트 후보 자동 추천 및 비교
- **상호작용**: 휠 확대/축소, 드래그 구간 확대, 더블클릭 초기화
- **테마**: 라이트 / 다크 모드 전환

## 핵심 기능

### 1. 로컬 우선 데이터 입력
- CSV 파일 선택
- CSV 텍스트 붙여넣기
- 샘플 데이터 즉시 로드
- 브라우저 내부 처리, 서버 업로드 없음

### 2. 자동 차트 추천
데이터 컬럼을 분석해 아래 차트 후보를 자동으로 구성합니다.

- 선 차트
- 영역 차트
- 막대 차트
- 산점도
- 파이 차트
- 도넛 차트
- 레이더 차트

각 차트는 단순히 숨기지 않고 아래 상태로 보여줍니다.

- **바로 보기**: 지금 데이터로 자연스럽게 표현 가능
- **확인 필요**: 표현은 가능하지만 해석 주의 필요
- **조건 부족**: 필요한 컬럼이 부족함

### 3. 대량 데이터 대응
- 전체 데이터를 기준으로 차트를 유지
- 표시 지점이 너무 많을 때는 균등 샘플링으로 렌더링
- 휠로 확대/축소
- 드래그로 특정 구간 확대
- 더블클릭으로 전체 범위 복귀

### 4. 데이터 이해 보조 패널
- 행 수 / 컬럼 수 / 추천 가능 상태 요약
- 컬럼 타입 추론 결과
- 표 미리보기

## 현재 상호작용 메모

현재 차트 확대는 **휠 확대/축소 + 드래그 확대** 중심입니다.

- 장점: 빠르게 전체 흐름과 특정 구간을 오갈 수 있음
- 한계: 트랙패드/마우스 환경에 따라 휠 줌이 다소 어색하게 느껴질 수 있음

다음 개선 후보:

- 드래그 확대를 기본 상호작용으로 더 강조
- 축/브러시 기반 구간 선택 UI 추가
- 키보드 기반 확대/축소 보강

## 데이터 처리 제한

- 브라우저에서만 처리됩니다.
- 기본 제한은 **1MB 또는 5,000행**입니다.
- 단순 CSV / TSV 중심 입력을 가정합니다.

## 프로젝트 의도

이 프로젝트는 단순 차트 뷰어보다,

> **“데이터를 넣었을 때 어떤 차트가 가장 잘 맞는지 빠르게 비교하고 설명해 주는 UI”**

를 만드는 데 목적이 있습니다.

즉, 차트를 그리는 것 자체보다도  
**입력 → 추천 → 비교 → 해석 확인** 흐름을 짧고 명확하게 만드는 데 집중했습니다.
