# 데이터 시각화 시뮬레이터 MVP

브라우저에서 서버 없이 데이터를 파싱하고 여러 차트 후보를 즉시 보여주는 로컬 우선 시각화 시뮬레이터입니다.

## 핵심 범위

- 샘플 데이터 선택
- 직접 입력/붙여넣기
- 로컬 CSV 파일 처리
- 컬럼 타입 추론
- 여러 차트 후보 카드 표시
- 부적합 차트의 더미/경고 상태 표시

## 제외 범위

- 서버 업로드
- 로그인/프로젝트 저장
- 코드 스니펫 또는 차트 설정 내보내기
- 대시보드 편집기

## 데이터 처리 제한

- 데이터는 브라우저에서만 처리됩니다.
- 기본 제한은 1MB 또는 5,000행입니다.
- MVP CSV 지원 범위는 단순 CSV/TSV 중심입니다.


## UX 고도화 방향

현재 버전은 복잡한 대시보드보다 사용자 친화적인 단계형 흐름을 지향합니다.

1. 데이터 선택
2. 데이터 이해
3. 추천 차트 먼저 보기
4. 대안 차트 비교

차트 후보는 `바로 보기`, `확인 필요`, `더미` 상태로 나뉘며, 맞지 않는 차트도 숨기지 않고 이유를 알려줍니다.
