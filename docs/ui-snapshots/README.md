# UI 스냅샷 기록

UI/디자인 변경을 점진적으로 진행할 때 화면 상태를 이미지로 남기는 폴더입니다.

## 규칙

- 파일명은 `YYYYMMDD-HHMM-설명.png` 형식을 사용합니다.
- 큰 레이아웃 변경, 디자인 검수 반영, 릴리즈 전 상태마다 스냅샷을 남깁니다.
- 스냅샷과 함께 어떤 변경을 확인한 것인지 커밋 메시지 또는 최종 보고에 적습니다.

## 현재 스냅샷

- `20260421-1929-baseline-friendly-workspace.png`: 1차 사용자 친화 레이아웃 상단 기준
- `20260421-1930-full-friendly-workspace.png`: 1차 사용자 친화 레이아웃 전체 기준
- `20260421-2241-redesigned-empty-workspace.png`: 앱형 레이아웃 입력 대기 상태
- `20260421-2242-redesigned-sample-workspace.png`: 앱형 레이아웃 샘플 CSV 로드 상태
- `20260421-2314-studio-empty-workspace.png`: 스튜디오형 레이아웃 입력 대기 상태
- `20260421-2314-studio-sample-workspace.png`: 스튜디오형 레이아웃 샘플 CSV 로드 상태
- `20260421-2326-studio-svg-empty-workspace.png`: 자체 SVG 차트 렌더러 적용 후 입력 대기 상태
- `20260421-2326-studio-svg-sample-workspace.png`: 자체 SVG 차트 렌더러 적용 후 샘플 CSV 로드 상태
